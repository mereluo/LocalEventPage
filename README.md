# LocalEventPage

Build with Next.js

## System Setup

1. Node version satisfies [here] (https://nextjs.org/docs/getting-started"); check your node version: `node -v`
2. `brew install nvm`
3. `npx create-next-app@latest <app-name>`
4. Run your app using: `npm run dev`
